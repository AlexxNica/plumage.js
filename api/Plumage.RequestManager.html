Class: RequestManager

Plumage. RequestManager

new RequestManager()

Singleton object. Call loadModel to load a Model, and have RequestManager keep a reference to the request so it can be cancelled if necessary.

Request manager also triggers 'message' on the Plumage.App for flash messages if the response contains a message.

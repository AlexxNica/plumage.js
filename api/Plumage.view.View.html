Class: View

Plumage.view. View

new View()

View class that renders to a DOM element. Very basic. Most of the time you'll want to use/extend ContainerView or ModelView.

CAUTION: Do not override render like you would with a Backbone.View. Override the template method onRender instead.

In addition to what Backbone.View does, Plumage.view.View adds:

  • default onRender that renders a handlerbars template with data from getTemplateData.
  • beforeRender and afterRender events and hooks.
  • deferRender flag which defers rendering until onShow is called. Good for tabs etc which aren't visible initially.
  • onShow and onHide hooks. These are called automatically by BaseController.

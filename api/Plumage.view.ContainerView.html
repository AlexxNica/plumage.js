Class: ContainerView

Plumage.view. ContainerView

new ContainerView()

View that contains subViews.

Implements composite pattern, i.e. calling render on ContainerView will call render on all it's subViews and their subViews etc, in depth first order.

Composite methods like render must call super to ensure recursion.

To add a new compositable method (eg updateModel) add it to the ContainerView subclass, and use callOrRecurse to propagate the call even when some subViews don't have the new method. This allows adding new methods without having to have every view implement them.

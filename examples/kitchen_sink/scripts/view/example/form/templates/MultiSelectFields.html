These fields let you pick a value from a list. Their common base class is Plumage.form.fields.Select

The set of options is taken from another model (the list model) that is also bound to the field.

Bind the listModel with Select.setListModel(), or use the Select.listRelationship option to have it automatically bind in the model tree.

Plumage.form.fields.MultiSelect

A basic html select with multiple="true". Hold ctrl/command to select multiple.

Plumage.form.fields.DropdownMultiSelect

A multi select using a bootstrap dropdown with checkboxes.

It has an optional select all button
